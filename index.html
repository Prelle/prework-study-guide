HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Header elements h1 - h6 are used to break the content up into logical sections.
  • The ul/li elements can be used to create an unordered list of items (like this one).

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border property determines the width and color of the border surrounding an element.
  • A box-shadow property can be added to create a drop shadow effect around an element's frame.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit: commits changes to the current branch
  • git pull: pulls changes from the repository to our local machine

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if/else statement can be used to control branching flow based on set conditions.
  • A for loop can be used to execute a block of code multiple times.
  • Functions can be created to encapsulate blocks of code for easy reuse.
